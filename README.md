### Prerequisites
Ensure you have the following installed:
- **Node.js** (>= 18.x)

### Installation
1. **Clone the repository**:
   ```sh
   git clone https://github.com/Shambhavigaur31/CodeBinge.git
   cd CodeBinge
   ```

2. **Install dependencies**:
   ```sh
   npm install
   # or
   yarn install
   ```

   ```sh

   ```

4. **Build for production**:
   ```sh
   npm run build && npm start
   ```
